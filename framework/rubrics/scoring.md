# Assessment Scoring Rubric

## Mastery levels (0–5)

| Level | Meaning |
|-------|---------|
| 0 | Not assessed / no exposure |
| 1 | Heard of it; needs guided help |
| 2 | Partial recall; frequent mistakes |
| 3 | Working knowledge; occasional gaps |
| 4 | Solid; can apply under mild pressure |
| 5 | Strong; can teach or troubleshoot others |

## Per-question scoring

- **Correct + confident + good reasoning** → full credit; may bump mastery +1
- **Correct but unsure** → credit; note for review
- **Partial** → half credit; stays on review queue
- **Wrong** → no credit; add misconception note; schedule revisit

## Difficulty calibration

Target **~60%** on mixed assessments. If session score:

- **> 80%** → increase difficulty or skip ahead in topic graph
- **40–80%** → stay at current level (ideal zone)
- **< 40%** → simplify, add scaffolding, revisit prerequisites

## Assessment mix (default)

Each session uses 2–4 formats drawn from:

1. CLI task (hands-on)
2. Scenario puzzle (diagnose/fix)
3. Short MCQ (fast retrieval)
4. Explain-in-your-own-words (depth check)

Placement uses all four. Regular lessons use 2–3.
